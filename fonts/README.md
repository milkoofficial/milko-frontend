# Custom Font Files

Place your custom font files here:

- `YourFont-Regular.woff2` (or .woff, .ttf, .otf)
- `YourFont-Bold.woff2` (or .woff, .ttf, .otf)

## Supported Formats (in order of preference):
1. **.woff2** - Best compression, modern browsers (recommended)
2. **.woff** - Good compression, wider browser support
3. **.ttf** - TrueType, universal support
4. **.otf** - OpenType, universal support

## Font File Naming:
You can name your files anything, but make sure to update the paths in `app/layout.tsx`.

Example names:
- `CustomFont-Regular.woff2`
- `CustomFont-Bold.woff2`
- Or: `font-regular.woff2`, `font-bold.woff2`
